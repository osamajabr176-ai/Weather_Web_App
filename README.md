📌 Overview
A modern, responsive bilingual weather application built with React that delivers real-time weather data using the OpenWeather API. The project focuses on clean UI design, internationalization, and real-world API integration, providing users with an interactive experience in both English and Arabic with automatic layout direction switching (LTR/RTL).

This project demonstrates practical frontend engineering skills including API handling, state management, localization, and real-time UI updates.

✨ Key Features
🌍 Real-time Weather Data using OpenWeather API
🌐 Bilingual Support (English 🇬🇧 / Arabic 🇸🇦)
🔄 Dynamic UI Direction (LTR / RTL switching)
📅 Live Date & Time Updates using Moment.js
🌡️ Displays temperature, weather condition, min/max values
⚛️ Built with React Hooks (useState, useEffect)
📡 API handling with Axios (with cancel token support)
🎨 Clean and responsive UI design

🧠What I Learned
Working with external REST APIs in real-time applications
Handling asynchronous data using Axios
Managing global state and side effects with React Hooks
Implementing multilingual support using i18next
Handling UI direction changes (RTL/LTR) dynamically
Improving performance with cleanup functions in useEffect

🛠️ Tech Stack
